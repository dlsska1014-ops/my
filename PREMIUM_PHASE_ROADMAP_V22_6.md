## 1차 개발 후보

### 영수증 자동 기록

- 영수증 1장 업로드
- 상호·결제일·총액 인식
- 중복 영수증 확인
- 처리 가계부와 분류 제안
- `맞다면 네라고 답해 주세요` 확인 후 저장
- 원본 이미지는 기본 미보관

필요 선행조건:

- 카카오 이미지 전송 방식 실제 채널 검증
- 개인정보 동의 문구
- OCR 공급자와 비용 상한
- 비동기 처리 및 실패 복구

### 반복 거래 자동화

- 반복지출 후보를 사용자에게 먼저 제안
- 확인한 항목만 반복 규칙으로 등록
- 실행 전 알림 또는 자동 실행 모드 선택
- 금액 변경 시 자동 저장하지 않고 확인 요청
- 중복 실행 방지 키와 실행 이력

### 고급 정산

- 동일 금액 분배
- 비율 분배
- 인원수 기준 분배
- 품목별 참여자 선택
- 최소 송금 횟수 계산
- 정산 완료 상태와 재정산 이력

### 자동 리포트

- 주간·월간 요약
- 의미 있는 변화가 있을 때만 발송
- 가계부별 수신 설정
- 중복 발송 방지
- 카카오 발송 정책·비용 확인 후 활성화

## 결제 연결 전 필수 조건

- 무료·프리미엄 권한표 확정
- 결제 성공·실패·취소·환불·해지 상태 모델
- 구독 만료 후 데이터 접근 원칙
- 개인정보처리방침·이용약관·환불정책 개정
- 결제 웹훅 서명 검증과 중복 처리 방지
- 관리자 수동 보정과 감사 이력
- 운영 비용 상한과 사용자별 사용량 제한

## 무료 기능으로 유지할 항목

- 거래 기록·수정·삭제
- 가계부 생성·참여·전환

- 데이터 백업·탈퇴·삭제
- 권한·보안·중복 저장 방지
