# V21.4.1-KAKAO-RESPONSE-ROUTING-HOTFIX

## 적용 목적
실제 카카오 그룹방에서 확인된 다음 문제만 최소 범위로 수정했습니다.

1. `내 예산`이 예산 조회가 아니라 일반 메뉴로 응답
2. `/단톡방 연결`의 줄바꿈이 문자 `\n`으로 노출
3. `/정산`이 구성원 정산이 아니라 월 요약으로 응답
4. 일반 응답마다 URL과 quickReplies가 반복 노출

## 원인
- 예산 명령 정규식에 `내 예산`, `우리 예산` 자연어가 빠져 있었습니다.
- 단톡방 안내 문자열 일부가 `.join("\\n")`을 사용해 실제 줄바꿈이 아니라 역슬래시 문자를 출력했습니다.
- `정산`이 `isSummaryCommand()`에 포함되어 월 요약 분기로 들어갔습니다.
- `kakaoText()`가 모든 응답에 quickReplies를 자동 부착했고, 여러 응답 생성 함수가 웹 URL을 기본 삽입했습니다.

## 수정 내용
- 앱 버전: `V21.4.1-KAKAO-RESPONSE-ROUTING-HOTFIX`
- `내 예산`, `우리 예산`, `예산 알려줘` 등을 예산 조회 명령으로 인식
- 문자형 `\\n`을 실제 줄바꿈으로 정규화하고 단톡방 연결 원본 문자열도 `\n`으로 수정
- `정산`을 월 요약에서 분리해 구성원별 납부액, 1인 기준액, 받을/보낼 금액, 송금 제안을 반환
- 일반 응답에서 URL 자동 제거
- `링크`/`주소` 명령에서만 `https://ttokttok-accountbook.com` 1개 반환
- 일반 응답의 `template.quickReplies` 기본 생성을 중단
- 사용자 명령 메뉴와 OpenBuilder 문서를 정식 용어 기준으로 정리

## 기대 응답

### `@똑똑한가계부 내 예산`
월 예산, 사용 금액, 남은 예산, 사용률을 반환합니다. 메뉴 안내나 URL은 붙지 않습니다.

### `/단톡방 연결`
실제 줄바꿈으로 연결 안내를 반환하며 `\\n` 문자가 표시되지 않습니다.

### `/정산`
월 요약이 아니라 1/N 정산 결과와 송금 제안을 반환합니다.

### `/요약`
수입·지출·잔액·상위 지출만 반환합니다.

### `/링크`
대표 주소 1개만 반환합니다.
