## 5. 카카오 로그인

```text
PUBLIC_BASE_URL=https://ttokttok-accountbook.com
KAKAO_REDIRECT_URI=https://ttokttok-accountbook.com/auth/kakao/callback
KAKAO_LOGIN_ENABLED=1
KAKAO_REST_API_KEY=<같은 카카오 앱의 REST API 키>
```

- [ ] 카카오 관리자센터 Web domain과 Redirect URI를 글자 하나까지 대조
- [ ] `www`, 끝 슬래시, `workers.dev`, 별도 미리보기 주소가 섞이지 않았는지 확인
- [ ] 설정이 맞기 전에는 `KAKAO_LOGIN_ENABLED=0` 유지
- [ ] 실제 카카오 로그인 후 기존 가계부 또는 새 사용자 시작 흐름으로 이동하는지 확인

`KOE006`은 카카오 관리자센터에 등록된 앱 설정과 요청값이 다를 때 카카오가 반환하는 오류입니다. Worker 코드만으로 관리자센터 등록값을 바꿀 수 없습니다.
