# V21.5 단계형 온보딩·예산 설정 번들 보고서

## 버전

```text
V21.5-GUIDED-ONBOARDING-BUDGET-BUNDLE
```

## 목적

처음 사용자가 카카오톡 안에서 안내 버튼과 짧은 입력만으로 가계부 생성·참여·예산 설정·사용자 이름 연결·첫 기록까지 완료하도록 개선했습니다.

## 구현 내용

### 1. 단계형 시작

- 가계부가 없는 사용자에게 “먼저 가계부를 만들어 보세요!” 안내
- 새 가계부 만들기, 초대코드 참여, 사용법 보기 제공
- 기존 사용자는 첫 기록, 예산 설정, 오늘 요약, 단톡방 연결, 내 이름 설정 제공

### 2. 카카오톡 가계부 생성·참여

- 가족 생활비, 부부·커플, 모임 회비, 여행 경비, 직접 입력
- 이름 입력 후 가계부·소유자 멤버십·초대코드 생성
- 초대코드로 기존 가계부 참여

### 3. 예산 설정

- 전체 월 예산
- 카테고리별 예산
- 지난달 예산 복사
- 금액 선택 또는 직접 입력
- 저장 전 명시적 확인
- 소유자·관리자만 변경 가능

직접 명령 예시:

```text
식비 예산설정 백만원
교통비 예산 설정 30만원
전체 예산설정 300만원
8월 식비 예산설정 80만원
```

### 4. 예산 현황

- 전체 예산
- 현재 사용 금액
- 잔여 예산
- 전체 사용률
- 카테고리별 사용액·잔여액·사용률

### 5. 사용자별 지출

- `내 이름 설정` 단계 제공
- 카카오 사용자와 가계부 표시명 매핑
- 거래 저장 결과에 결제자 표시
- 날짜별 요약에 결제자별 지출 합계 표시

### 6. 날짜별 요약

지원 범위:

```text
오늘
어제
이번 주
지난 주
이번 달
지난 달
특정 월
특정 날짜
```

요약 항목:

- 수입
- 지출
- 거래 건수
- 카테고리별 지출
- 결제자별 지출

### 7. 응답 정책

- 일반 거래 저장: 링크 없음, quickReplies 없음
- 남은예산 일반 조회: quickReplies 없음
- 미인식 안내: quickReplies 없음
- 시작·가계부 생성·예산 설정·이름 설정 등 단계형 흐름에서만 quickReplies 사용
- 가계부 생성, 예산 완료, 상세 요약 같은 자연스러운 지점에서만 홈페이지 CTA 제공
- 동일 CTA는 사용자·가계부·종류별 24시간 반복 억제

### 8. 상태 저장

- 별도 스키마 추가 없음
- 기존 `accountbook_settings` 사용
- 사용자와 그룹방을 조합한 상태 키 사용
- 단계 상태 만료 시간 30분
- OpenBuilder 컨텍스트를 영구 상태로 사용하지 않음

## 변경하지 않은 범위

- Supabase 테이블 구조
- 기존 거래 저장 테이블과 권한 구조
- `/my`, `/app` 사용자 화면 핵심 동작
- 카카오 로그인
- 중복 저장 방어
- 레이트리밋
- 기존 공개 도메인
