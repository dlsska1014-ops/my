# v17.1 전체 흐름 안정화 가이드

## 1. 기본 흐름
`/my` → 홈 입력 → 설정 → 캘린더 → 분석 → 백업/참여자/단톡방 연결로 이동합니다.

## 2. 단톡방 연결
카카오 챗봇이 단톡방에 들어가 있어야 합니다.
단톡방에서 `단톡방 연결 초대코드`를 입력하면 연결됩니다.
웹 사용자키와 카카오 사용자키가 달라도 초대코드가 맞으면 연결됩니다.

## 3. 캘린더
날짜를 누르면 하단에 해당 날짜 기록이 나오며 수정/삭제할 수 있습니다.

## 4. 분석
소비몬, 패턴, 고정비, 카페/배달/쇼핑/구독, 분류별 지출/건수를 확인합니다.

## 5. 키워드 안내
개인 가족 이름 예시는 제거했고, 일반 예시만 사용합니다.
