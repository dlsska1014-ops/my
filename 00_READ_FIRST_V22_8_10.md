# V22.8.10 먼저 읽기

## 무엇을 적용하나요?

ZIP의 `src/index.js`를 현재 Cloudflare Worker 코드와 **전체 교체**합니다. 부분 붙여넣기는 하지 않습니다.

## SQL도 실행하나요?

아니요. 이번 버전은 신규 테이블·컬럼·RPC·인덱스를 만들지 않습니다. 과거 SQL도 다시 실행하지 않습니다.

## 무엇이 빨라졌나요?

- 홈 데이터 요청: 16회 → 9회
- `/my` 진입 요청: 7회 → 4회
- 개인 홈 HTML: 106,841 bytes → 24,490 bytes
- 공통 CSS·JavaScript는 버전형 정적 경로로 분리해 재사용
- 가계부·참여자·설정의 반복 조회를 묶음 조회로 변경

## ZIP은 어떻게 정리했나요?

과거 개별 로그, 중복 체크섬, 이미지, 빌드 폴더, 원본 학습 ZIP은 제외했습니다. 버전별 내용은 `CHANGELOG.md`, `HISTORY_INDEX.md`, SQL 이력은 `SQL_HISTORY.md`에 남겼습니다.

4. 홈 금액·예산·참여자·최근 기록 확인
