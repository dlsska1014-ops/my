# V19.8.4-BIZ-FOOTER-ONLY-HOTFIX

## 목적
카카오 비즈니스 심사에서 `/my` 첫 화면에도 사업자 정보가 명확히 보여야 한다는 피드백을 반영한 최소 핫픽스입니다.

## 변경 내용
- `/my` 비로그인 첫 화면에 사업자 정보 카드를 명시적으로 노출
- `/my` 새 가계부 만들기/초대코드 참여 첫 화면에 사업자 정보 카드 노출
- `/my`가 로그인 후 `/app`으로 이동되는 실제 첫 화면에도 사업자 정보 카드 노출
- 기존 전체 HTML 하단 사업자 푸터 자동 삽입은 유지
- 사업자 정보 카드가 있어도 하단 푸터가 생략되지 않도록 중복 판정 조건 보정
