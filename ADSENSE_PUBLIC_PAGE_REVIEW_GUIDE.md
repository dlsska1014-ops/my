# AdSense 공개 페이지 운영 가이드

## 심사 전
1. `ADSENSE_PUBLISHER_ID=ca-pub-실제번호` 설정
2. `ADSENSE_ENABLED=0`, `ADSENSE_AUTO_ADS=0` 유지
3. `/ads.txt`, `/sitemap.xml`, `/robots.txt` 확인
4. 공개 페이지의 모바일 탐색, 오탈자, 링크 확인
5. `/privacy`, `/terms`, `/cookies`, `/contact` 실제 운영 내용 최종 검토
6. 공식 문의 이메일이 있으면 `PUBLIC_SUPPORT_EMAIL` 설정

## 심사 중
- 공개 페이지와 핵심 서비스 구조를 크게 변경하지 않음
- 빈 광고 영역이나 공사 중 문구를 표시하지 않음
- 로그인·거래 입력·예산 설정·오류 화면에 광고를 삽입하지 않음

## 승인·카카오 런칭 안정화 후
- 공개 가이드 하단 또는 분석 콘텐츠 중간에 수동 슬롯 1개부터 시작
- Auto Ads는 초기 비활성 유지
- 페이지 속도, 이탈률, 오클릭과 첫 기록 완료율을 비교
- 개인정보처리방침과 쿠키 동의 요구사항을 대상 지역에 맞게 적용
