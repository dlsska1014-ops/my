# V21.5.2 자연어 의도·스마트 CTA 핫픽스

내부 버전: `V21.5.2-NATURAL-INTENT-SMART-CTA-HOTFIX`

## 목적

V21.5.1 지연 문제 해결 이후 실제 사용자 발화에서 확인된 두 가지 공백을 보완합니다.

1. `닉네임수정`, `닉네임 변경`, `표시명 변경` 같은 일상 표현이 인식되지 않음
2. `초대코드`처럼 웹 관리가 자연스러운 명령에서도 관리 링크가 누락됨

## 변경 내용

- 이 가계부 표시명 변경 동의어 확대
- `닉네임 인남으로 변경` 같은 한 문장 직접 변경 지원
- 초대·구성원 관련 동의어 확대
- 초대코드 응답에 초대 문구와 가계부 범위가 포함된 `참여자·초대 관리` 링크 1개 제공
- 도움말에 `닉네임수정`과 `내 이름 설정` 추가
- 예산 변경, 가계부 전환, 단톡방 연결, 정산, 날짜별 지출 질문의 안전한 동의어 확대
- 미인식 응답이 닉네임·초대 의도를 감지하면 해당 기능 예시를 우선 안내

## 유지 원칙

- OpenBuilder 시나리오명·블록명·스킬 URL 변경 없음
- 모든 동적 자연어는 폴백 → `/skill` 경로 유지
- 일반 거래 저장 응답에는 URL과 quickReplies를 붙이지 않음
- 초대코드처럼 사용자가 관리 의도를 명시한 경우에만 관련 URL 1개 제공
- Supabase 스키마·가계부 권한·거래 저장 구조 변경 없음
