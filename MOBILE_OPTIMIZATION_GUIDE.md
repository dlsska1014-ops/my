# v18.5 모바일 최적화 가이드

## 적용 방향
PC 화면을 기준으로 기능은 하나로 유지하고, 모바일에서는 CSS 반응형으로 보기 좋게 조정했습니다.

## 모바일에서 달라지는 점
- 상단 고정 메뉴가 안전영역까지 고려됩니다.
- 하단 빠른 메뉴가 5개 탭으로 정상 배치됩니다.
- 입력폼은 모바일에서 한 줄씩 크게 표시됩니다.
- 기록표/캘린더처럼 넓은 영역은 좌우 스크롤 안내가 표시됩니다.
- 주요 화면은 하단 메뉴와 겹치지 않도록 여백이 추가됩니다.
