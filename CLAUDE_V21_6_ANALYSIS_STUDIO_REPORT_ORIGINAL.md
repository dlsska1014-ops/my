# V21.6.0 ANALYSIS STUDIO 리포트

## 배경

기존 `/my/analysis`(종합분석)는 서버 렌더링 표 중심이라 토스·편한가계부·뱅크샐러드 대비
분석 도구가 부족하고, 조건을 바꿀 때마다 새로고침이 필요했습니다. 이번 버전은 분석 화면을
클라이언트 사이드 인터랙티브 대시보드(분석 스튜디오)로 교체합니다.

## 구현 범위

### 라우트

- `GET /my/analysis` → 신규 분석 스튜디오 (기본)
- `GET /my/analysis?view=report` → 기존 종합 리포트 유지 (소비몬·주간 리포트·반복지출·큰지출 체크)
- `GET /my/analysis/app.js` → 클라이언트 번들(워커 내 `insightClientMain` 함수를 직렬화해 서빙, 캐시 10분)

### 데이터 흐름

- 서버는 로그인·가계부 권한 확인 후 최근 12개월 거래를 슬림 배열
  `[날짜, 수입여부, 금액, 분류, 결제수단, 메모, 입력자]`로 임베드
- 필터·집계·차트는 전부 브라우저에서 즉시 계산 (서버 왕복 없음)
- 외부 라이브러리 0개, 인라인 SVG 차트 — CSP·오프라인·속도 부담 없음

### 필터 (모두 자유 조합, 즉시 반영)

- 기간: 이번 달 / 지난달 / 최근 7·30·90일 / 최근 6·12개월 / 직접 날짜 범위
- 유형: 지출 / 수입 / 전체
- 분류·결제수단·구성원: 다중 선택 칩
- 금액대(최소~최대), 텍스트 검색(메모·분류·결제수단·입력자)
- 활성 필터 칩에서 개별 해제, 초기화 버튼, URL에 상태 저장(공유·새로고침 유지)

### 시각화

- KPI 5종: 지출·수입·수입-지출·하루평균·건수 + 직전 동일 기간 대비 증감률
- 지출 흐름: 기간 길이에 따라 일/주/월 자동 버킷, 호버 툴팁, 최대 구간 직접 라벨,
  막대 클릭 시 해당 구간으로 드릴다운, "표로 보기" 대체 뷰
- 분류 도넛: 상위 6개 + 그 외, 분류별 색상은 전체 기간 순위 기준으로 고정(필터에도 색 유지),
  범례 클릭으로 필터 토글
- 요일 패턴 / 결제수단별 / 구성원별: 단일 색상 순위 막대, 클릭 필터
- 이번 달 예산: 전체 게이지 + 분류별 사용률 (80% 경고색, 100% 초과색)
- 큰 금액 TOP 8, 날짜별 그룹 기록 목록(더 보기 페이지네이션), 필터 결과 CSV 내려받기

### 색상·접근성

- 카테고리 팔레트 8색은 색각이상 검증 도구로 확인 (인접 색 CVD ΔE 24.2 ≥ 12 통과)
- 대비 부족 색(아쿠아·옐로·마젠타)은 항상 라벨+금액 텍스트 병기, 모든 차트에 표 대체 뷰
- 텍스트는 항상 잉크 색, 시리즈 색은 마크(막대·점)만 사용
- 툴팁은 호버+키보드 포커스 동일 제공, 값은 툴팁 없이도 표에서 접근 가능

## 유지한 것

- 기존 종합 리포트 전체 (`?view=report`)
- `/my` 저장 흐름, 권한/범위 제한, Supabase 테이블 구조 (마이그레이션 없음)
- 카카오 스킬·OpenBuilder 블록·발화 변경 없음
