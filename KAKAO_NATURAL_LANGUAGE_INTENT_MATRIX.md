# 카카오 자연어 의도 지원표 — V21.5.2

## 내 이름·닉네임

단계형 시작:

- `닉네임수정`
- `닉네임 변경`
- `별명 설정`
- `표시명 변경`
- `내 이름 바꾸기`
- `결제자 이름 변경`

한 문장 직접 변경:

- `닉네임 인남으로 변경`
- `내 이름 인남으로 바꿔`
- `인남으로 표시명 변경`

저장 대상은 전체 카카오 프로필명이 아니라 현재 가계부의 표시명입니다.

## 초대·참여

초대코드 확인·공유:

- `초대코드`
- `초대코드 보여줘`
- `구성원 초대`
- `가족 초대`
- `초대 방법`

응답에는 초대코드, `가계부 참여 ABC123` 문장, `참여자·초대 관리` URL 1개가 포함됩니다.

참여:

- `초대코드로 참여`
- `가계부 참여 ABC123`

## 링크 정책

- 일반 거래, 예산 조회, 단순 미인식: URL 없음
- 초대코드 확인: 참여자·초대 관리 URL 1개
- 상세 분석 CTA: 동일 사용자·가계부·종류 기준 24시간 반복 억제
- 사용자가 `웹으로 보기`, `관리 화면`, `상세보기`를 명시하면 대표 웹 링크 제공
