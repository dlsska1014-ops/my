# Full Audit Report — V21.1

## 변경 요약

V21.0 도메인 후보판 위에 신규 그룹 챗봇 제작 흐름과 대량 트래픽 대응 기준을 추가했다. 공개 URL은 환경변수 기반으로 고정하며, 개인 계정명/개인 이메일/개인 Worker 주소가 심사 화면에 섞이지 않도록 점검 경로를 추가했다.

## 추가된 기능

- 신규 그룹 챗봇 제작 가이드 화면
- OpenBuilder 시작 블록 기준
- 개인 주소 제거 점검 화면
- 대량 트래픽 준비 화면
- 카카오 신규 봇 설정 JSON
- 운영센터 진입 메뉴 보강

## 보존된 기능

- 사용자 홈/앱/스킬 저장 로직
- 권한/가계부 scope
- 여러 가계부 생성/참여
- 빠른입력 분류
- 중복/트래픽 방어
- 예산 알림
- 밈 콘텐츠센터
- 사업자 푸터
- Supabase 테이블 구조

## 위험 관리

- `/skill`은 카카오 서버 IP 단위로 제한하지 않고 `botUserKey` 단위로 제한한다.
- 중복 저장 방어와 카카오 재전송 방어는 유지했다.
- 그룹 챗봇 신규 제작 시 기존 개인 링크를 복사하지 않는다.
- 공개 주소는 `PUBLIC_BASE_URL` 기준으로만 안내한다.
