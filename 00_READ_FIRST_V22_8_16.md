# V22.8.16 먼저 읽기

검증된 `src/index.js`를 현재 Cloudflare Worker 코드와 **전체 교체**합니다. 부분 붙여넣기는 하지 않습니다.

## 변경 범위

- 카카오 챗봇 수정(edit) 플로우를 V4 파서·상태머신으로 전면 교체
- 재현 버그(`수정 03번` → `지출자 변경` → 같은 메뉴 반복) 구조적 차단
- 세션 중 입력은 다른 어떤 핸들러로도 가지 않는 라우팅 순서 재구성
- 동일 봇 응답 2회 연속 전송을 물리적으로 차단하는 전송 직전 가드
- 실패 3회·총 8턴 상한, 취소·도움말 상시 인식, 5분 세션 TTL
- 선택형 값(지출자·결제수단·분류·날짜)은 실데이터 기반 숫자 옵션
- `삭제 NN번` 즉시 삭제 + `복구 NN번` 되돌리기(24시간)
- 저장 응답·메뉴 문구 교체, `결제자` → `지출자` 용어 통일
