## V22.8.20 우선 · 운영 성능 계측과 DB 인덱스 판정

- 실제 `/my`, `/app`, `/my/households` p50·p95 확인
- Supabase 느린 쿼리, 행 수, 실행 계획 확인
- `/app` 9회와 홈 35KB 기준의 운영 유지 여부 확인
