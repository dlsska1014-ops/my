# V22.4.2 공개 콘텐츠·정책·UI 안정화 보고서

## 버전
`V22.4.2-PUBLIC-CONTENT-POLICY-UI-STABILITY-BUNDLE`

## 변경 범위
- 공개 페이지에서 내부 구현 용어(OpenBuilder 등) 제거
- 서비스 소개 문구를 개인·가족·부부·모임·여행 사용성과 쉬운 입력·보편화 중심으로 재작성
- 광고 위치에 관한 과도한 공개 약속 제거, 개인정보·쿠키 정책은 Google 공식 요구사항 중심으로 정리
- `/site-map` 사용자용 HTML 사이트맵 추가
- `/sitemap.xml`은 검색엔진용 XML을 유지하며 `/sitemap.xsl`로 브라우저 가독성 개선
- 공개 푸터의 사이트맵 링크를 XML이 아닌 사용자용 `/site-map`으로 변경
- 사업자 정보 푸터를 가로형·자동 줄바꿈 구조로 압축해 높이 감소
- About 페이지의 중복 사업자 정보 카드 제거
- 내부 링크는 같은 창, 외부 공식 정책 링크만 새 창으로 열도록 정리
- 공개 페이지 SEO 메타(robots, theme-color, og:site_name, locale) 보강

## 유지 범위
- V22.4.1 계정·가계부 안정화
- 카카오 `/skill`, OpenBuilder 설정, NLU, 거래·예산·권한·중복 방어
- Supabase 스키마
- AdSense 실제 광고 기본 비활성 정책
