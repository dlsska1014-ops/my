- [ ] `/identity-audit`에서 웹 로그인·카카오 챗봇 계정 분리 여부 확인
- [ ] 기억하지 못하는 가계부는 거래 건수 확인 전 삭제하지 않음

## Worker 배포

- [ ] OpenBuilder와 Supabase SQL은 변경하지 않음

## 직접 채팅 테스트

- [ ] 선택 상태가 없는 기존 사용자가 `시작` 입력 시 가계부 목록/확인 표시
- [ ] 선택 전 `초대코드` 입력 시 임의 코드가 바로 노출되지 않음
- [ ] 번호 또는 이름으로 선택 후 `초대코드`가 해당 가계부 기준으로 표시
- [ ] 거래 저장 응답에 `가계부: ...` 표시
- [ ] 가계부가 2개 이상이면 `가계부 전환`으로 선택 가능
- [ ] 가계부가 전혀 없는 신규 계정은 생성·참여 안내만 표시

## 단톡방 테스트

- [ ] 연결되지 않은 방에서 자연어 거래가 저장되지 않음
- [ ] 연결되지 않은 방에서 owner/admin 가계부 목록 표시
- [ ] 연결되지 않은 방의 `초대코드`가 임의 가계부 코드를 노출하지 않음
- [ ] `단톡방 연결 ABC123` 후 연결 가계부 이름 표시
- [ ] 연결 후 거래가 해당 가계부에만 저장되고 응답에 이름 표시
- [ ] 다른 방은 별도 연결 상태를 유지

## 삭제·정리 테스트

- [ ] 테스트 가계부 삭제 전 백업
- [ ] 삭제한 가계부의 그룹 연결이 남지 않음
- [ ] 삭제한 가계부의 사용자 선택 포인터가 남지 않음
- [ ] 다음 `시작`에서 남은 가계부 선택 또는 생성 안내 표시

- [ ] 통합 후 웹과 카카오톡에서 동일 가계부 목록 확인
- [ ] 통합 후 거래 총건수·owner 수·초대코드 유지 확인
