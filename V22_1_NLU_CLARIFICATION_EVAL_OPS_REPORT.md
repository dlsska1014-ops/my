# V22.1 NLU Clarification · Evaluation · Operations Bundle

## 목적
- 외부 LLM 비용 없이 한국어 자연어 의도 범위를 확대합니다.
- 짧고 모호한 표현을 임의 실행하지 않고 한 번의 선택형 확인으로 전환합니다.
- 합성 평가 데이터와 어려운 혼동 문장을 자동 회귀시험에 포함합니다.

## 코드 변경
- Intent Registry를 21개 의도로 확대
- 한국어 오타/축약/말투 정규화 보강
- `이름 변경`, `초대`, `예산`, `가계부`, `식비 50만원` 명확화 응답
- `/nlu-detect.json?q=` 진단 경로 추가
- NLU 감지 함수 named export 추가(로컬 평가 전용)
- OpenBuilder 시나리오명/블록명/스킬 URL 변경 없음
- Supabase 기존 스키마 변경 없음

## 비용 원칙
- 외부 LLM/Workers AI 호출 0
- 원문 발화 영구 저장 기본 0
- 선택 SQL은 시간당 집계값만 저장하며 적용하지 않아도 됨

## 평가 데이터 주의
`nlu_eval_dataset_v221.jsonl`은 실제 사용자 로그가 아닌 합성·수동 혼동 사례입니다.
자동 회귀 방지에는 유용하지만 실제 운영 인식률을 증명하지는 않습니다. 운영 후 익명 집계와 카카오 학습대기 발화로 보완해야 합니다.
