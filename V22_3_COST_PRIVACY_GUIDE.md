# V22.3 비용·개인정보 제어

기본 배포만으로 추가 DB 저장 비용은 발생하지 않는다. 영구 집계는 환경변수를 켠 경우에만 동작한다.

- 시간대별 집계: 요청 1건당 RPC 1회가 추가되므로 대규모 운영 전 샘플링/큐 집계로 확장 검토
- 실패 샘플: 폴백·확인·오류에만 선택적으로 저장
- 원문 저장 금지: 전화번호·이메일·URL·코드·긴 숫자를 마스킹
- 사용자 키·가계부 ID 저장 안 함
- 기본 보존기간 14일

30만 사용자 단계에서는 집계 RPC를 매 요청마다 호출하기보다 Cloudflare Analytics Engine, Queues 또는 배치 집계로 이전하는 것을 검토한다. 이번 버전은 실제 사용량을 측정하기 위한 저위험 운영 기반이다.
