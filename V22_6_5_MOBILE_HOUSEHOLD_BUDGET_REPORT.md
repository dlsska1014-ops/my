# V22.6.5 모바일·가계부 관리·예산 핫픽스 보고서

적용 버전: `V22.6.5-MOBILE-HOUSEHOLD-BUDGET-HOTFIX`

## 결론

제보된 여섯 화면과 관련 사용자 흐름을 코드·데이터·모바일 렌더링 기준으로 다시 점검했다. 홈 메뉴, 개인 비밀번호 복귀, 첫 가계부 생성, 템플릿 이름, 가계부별 관리, 영구 삭제, 참여자 카드, 최근 내역 필터, 수입·예산 계산, 고정 하단 메뉴 겹침을 수정했다.

자동 검증 기준으로 제한 사용자 대상 무료 베타 릴리스 후보에 해당한다. 운영 Supabase/RLS, 실제 카카오 OAuth·OpenBuilder, iPhone Safari·Android Chrome, 유료 결제는 로컬 모의 환경만으로 확정할 수 없으므로 최종 체크리스트를 거쳐야 한다.

## 새 가계부 흐름

1. 카카오 닉네임이 내 표시명에 채워진다.
2. 사용자가 가계부 이름을 직접 확인·수정한다.
3. 개인 비밀번호가 없으면 6자리 이상으로 두 번 입력한다. 이미 있으면 현재 비밀번호를 확인한다.
4. 가계부를 생성하고 생성자를 `owner`로 연결한다.
5. 생성 완료 화면에서 초대코드와 참여자 관리로 바로 이어진다.

개인 비밀번호는 가계부 구성원에게 공유하는 공용 암호가 아니라 로그인 복구와 민감 작업 확인을 위한 사용자별 암호다.

## 가계부 관리와 삭제

- 가계부 카드에서 열기·옵션·참여자·초대를 바로 선택한다.
- owner/admin은 이름을 변경할 수 있다.
- owner만 영구 삭제를 실행할 수 있다.
- 삭제 전 가계부 이름 일치, 개인 비밀번호 확인, 참여자·기록 삭제 동의가 모두 필요하다.
- 삭제 시 가계부의 거래, 참여자 연결, 예산·정기항목·분류, 가계부별 설정, 단톡방 연결과 선택 상태를 정리한다.
- CSV 백업 경로를 삭제 영역에 함께 제시한다.

## 수입·예산 계산 기준

- 실제 수입: 해당 월의 `income` 거래 합계이며 수입 분류별로 표시한다.
- 예상 수입: `__income:수입종류` 계획 항목의 합계다.
- 지출 예산: 일반 분류별 예산의 합계다.
- 남은 지출 예산: 지출 예산 합계 - 실제 지출이다.
- 실제 여유금: 실제 수입 - 실제 지출의 단순 차액이다.
- 이전 버전의 `__total` 값은 분류별 예산이 없을 때만 호환용으로 사용하며 새 계획 저장 시 제거한다.

월 전체 금액을 다시 직접 입력하는 중복 UI는 현재 사용자 화면에서 제거했다.
