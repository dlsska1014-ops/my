# V22.5.2 카카오 대표 명령어 설정

## 원칙

- 기존 OpenBuilder 시나리오명과 블록명은 변경하지 않습니다.
- 대표 명령어는 사용 빈도가 높은 10개만 노출합니다.
- 거래 기록은 `/기록`이 필수가 아니며 자연어 입력이 기본입니다.
- 가계부가 선택·연결되지 않은 상태에서는 초대코드·거래·예산을 임의 가계부로 처리하지 않습니다.
- 대표 명령어는 모두 기존 통합 Skill URL `https://ttokttok-accountbook.com/skill`로 전달합니다.

## 등록할 대표 명령어 10개

|순서|표시명|실행 문장|설명|
|---:|---|---|---|
|1|시작|시작|가계부 선택·생성·참여부터 시작|
|2|기록|기록|기록 예시 확인; 뒤에 내용을 쓰면 저장|
|3|오늘기록|오늘 기록|오늘 기록과 수정 번호 확인|
|4|남은예산|남은예산|예산·사용액·잔여액·사용률 확인|
|5|예산설정|예산 설정|전체·카테고리 예산 단계형 설정|
|6|요약|요약|현재 가계부의 이번 달 요약|
|7|정산|정산|보낼 돈·받을 돈 확인|
|8|가계부전환|가계부 전환|사용할 가계부 목록과 선택|
|9|초대코드|초대코드|선택한 가계부의 초대코드 확인|
|10|단톡방연결|단톡방 연결|현재 그룹방 연결 안내|

관리자센터가 명령어 앞의 `/`를 자동으로 표시한다면 **표시명과 실행 문장에는 슬래시 없이** 입력합니다. 직접 전체 명령을 입력하는 형식이라면 `/시작`, `/기록`처럼 입력합니다.

## 보조 명령어

- `/도움말` → 전체 사용법
- `/내이름설정` → 가계부 표시 이름 설정
- `/수정` → 번호 기반 수정·삭제 안내

## 공통 블록 설정

- 기존 시나리오명·블록명 유지
- ML/ML+: OFF
- 파라미터·엔티티·컨텍스트: 없음
- OpenBuilder 고정 답변: 없음
- 스킬데이터 사용: ON
- Skill URL: `https://ttokttok-accountbook.com/skill`
