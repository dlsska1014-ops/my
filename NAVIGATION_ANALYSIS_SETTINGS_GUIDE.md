# 메뉴/분석/설정 UX 가이드

## 메뉴
좌측 메뉴는 details 기반이라 접기/펼치기가 가능합니다.
홈, 분석, 캘린더, 설정으로 어디서든 이동할 수 있습니다.

## 분석
`/my/analysis`에서 월 지출, 수입, 하루 평균, 최다 분류, 무지출일, 분류별 지출/건수를 확인합니다.

## 캘린더
`/my/calendar`에서 일별 지출금액과 기록 건수를 확인합니다.

## 설정
수입/예산은 행 추가로 필요한 항목만 관리합니다.
드롭다운을 선택하면 직접입력 칸에 반영되며, 직접입력도 가능합니다.
키워드는 하단 접이식 2열 카드로 관리합니다.
