# V22.8.13 다크모드 전체 보정 보고서

## 운영 확인에서 발견한 문제

- 다크모드 예산 화면에 밝은 계획·안내 배경이 남음
- 고급 정산의 참여자·품목·안내 영역에 밝은 배경이 남음
- 내 설정의 카드와 키워드 편집기에 흰 배경과 흰 글자가 겹침
- 파란 링크 버튼에 밝은 파란 글자가 적용되어 작은 글자 대비가 부족함
- 라이트 분석의 비활성 필터 글자가 4.5:1 기준보다 낮음

## 적용

- 예산·정산·내 설정에 화면 전용 body 범위를 부여
- 레거시 카드·안내·키워드 칩·개수 배지·추가·삭제 컨트롤을 다크 표면 토큰으로 연결
- 주 버튼 링크는 흰 글자, 보조 버튼 링크는 다크 표면 위 강조색으로 분리
- 라이트 분석의 비활성 세그먼트 글자를 더 짙은 공통 토큰으로 교정
- 수정 CSS를 `/assets/accountbook-shell-v22813.css`로 분리해 immutable 캐시 충돌 방지
