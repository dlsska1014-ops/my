# V22.2 NLU + 분석 스튜디오 UI/UX 통합 리포트

## 검토 결론

Claude 패키지는 V21.5.2 위에 `/my/analysis` 분석 스튜디오만 추가한 독립 변경이었습니다. 기존 V22.1 NLU·평가·운영 기능을 유지한 채 분석 화면 변경만 선택 병합했습니다.

## 반영

- `/my/analysis`: 새 분석 스튜디오
- `/my/analysis?view=report`: 기존 종합 리포트 유지
- 기간·유형·분류·결제수단·구성원·금액·검색어 조합 필터
- KPI, 추이, 분류 도넛, 요일·결제수단·구성원, 예산, 큰 금액, 거래 목록
- 필터 URL 유지, 표 대체 보기, CSV 내려받기, 모바일 반응형
- 외부 차트 라이브러리 및 외부 AI 호출 없음

## 추가 보완

Claude 원본의 `getMyPageContext()` 재사용은 분석 화면에서 사용하지 않는 캘린더·통계까지 조회했습니다. V22.2에서는 회원·예산·12개월 거래 3개 요청만 병렬 실행하도록 변경했습니다.

- 분석 진입 DB 왕복 및 계산 축소
- 최신 9,000건 초과 시 잘림 안내
- CSV 수식 주입 방지
- app.js `nosniff` 헤더 추가
- V22.1 NLU 4,738건 평가와 스킬 지연 최적화 유지

## 변경하지 않음

- OpenBuilder 시나리오·블록·발화
- `/skill` URL
- Supabase 스키마
- 거래 저장·권한·중복 방지
- 카카오 온보딩·예산·자연어 Intent Registry
