# 단톡방-가계부 연결 가이드

## 목적
카카오 그룹 채팅방의 `botGroupKey`를 가계부와 연결해서, 같은 방에서 입력한 거래를 같은 가계부에 저장합니다.

## 연결 명령
그룹방에서 아래처럼 입력합니다.

```text
단톡방 연결 ABC123
```

ABC123은 가계부 초대코드입니다.

## 연결 후 사용
```text
점심 12000원 국민카드
요약
남은예산
오늘 기록
```

## 저장 위치
```text
accountbook_settings
key = kakao_group_links
```

## 주의
- 그룹방 식별키가 없는 1:1 채팅에서는 연결할 수 없습니다.
- 권한이 pending/viewer/blocked인 사용자는 연결할 수 없습니다.
- 그룹방에 연결되어도 사용자별 권한은 계속 확인합니다.
