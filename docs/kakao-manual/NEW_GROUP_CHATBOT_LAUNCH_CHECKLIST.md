# 신규 그룹 챗봇 제작 체크리스트

## 주소 원칙

- `PUBLIC_BASE_URL` 기준으로 새 챗봇을 만든다.
- 개인 계정명, 개인 이메일, 개인 Worker 주소는 카카오 블록/설정/심사 URL에 넣지 않는다.
- 기존 테스트 블록을 복사할 때 URL이 남아 있는지 확인한다.

## 카카오 관리자센터 블록

1. 봇 입장 블록
2. 도움말 블록
3. 가계부 연결 블록
4. 기록 저장 블록
5. 조회 블록
6. 예산 블록
7. 밈 카드 블록
8. 폴백 블록

## Skill URL

```text
${PUBLIC_BASE_URL}/skill
```

## 안전 문구

```text
가계부적은 봇에게 직접 보낸 명령어만 처리합니다.
기록 제공을 위해 명령어와 가계부 데이터를 저장하며 보관·삭제 기준은 도움말에서 확인할 수 있습니다.
```

## 금지 문구

```text
단톡방 대화를 읽어서 자동 분석합니다.
채팅 내용을 학습해 소비 패턴을 알아냅니다.
```
