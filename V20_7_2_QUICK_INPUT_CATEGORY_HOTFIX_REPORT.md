# V20.7.2 QUICK INPUT CATEGORY HOTFIX

## 목적
`/app` 빠른 입력창이 카카오 챗봇처럼 분류되지 않는 문제를 보정한 최소 핫픽스입니다.

## 수정 내용
- `/app` 한 줄 빠른 입력에서 금액·날짜·결제수단·메모·분류를 함께 채우도록 보강
- `raw_text` hidden field를 추가해 서버 저장 시 원문 기준 분류/결제수단 보정 가능
- `/admin/transactions`와 `/my/transactions` 직접 저장도 카카오 챗봇 저장과 같은 사용자 설정 기반 분류 키워드/결제수단 우선 반영
- 자주 나온 빠른입력 예시(쌀, 시리얼, 김자반, 마시멜로우, 어린이식기, 풋샴푸, 온열안대 등)에 대한 기본 분류 키워드 보강

## 유지한 내용
- `/my`, `/app`, `/skill` 핵심 저장 흐름
- 가계부 범위/권한 체크
- 여러 가계부 생성/참여
- 메뉴/밈 폴리시 핫픽스
- 예산 알림, 트래픽/중복 방어, 하단 사업자 푸터
- Supabase 테이블 구조
