# 카카오 대표 명령어 V22.5.4

카카오 관리자센터에는 아래 문구를 그대로 등록합니다. 표시 문구와 전송 문구는 동일합니다.

| 순서 | 대표 명령어 | 설명 |
|---:|---|---|
| 1 | 시작 | 입력하지 않음 |
| 2 | 새 가계부 만들기 | 입력하지 않음 |
| 3 | 초대코드로 참여 | 입력하지 않음 |
| 4 | 단톡방 연결 | 현재 단톡방을 선택한 가계부에 연결해요. |
| 5 | 기록 방법 | 입력 예시를 확인해요. 거래는 말하듯 바로 입력할 수 있어요. |
| 6 | 오늘 기록 보기 | 입력하지 않음 |
| 7 | 예산 설정 | 입력하지 않음 |
| 8 | 남은 예산 | 입력하지 않음 |
| 9 | 이번 달 요약 | 입력하지 않음 |
| 10 | 가계부 전환 | 앞으로 기록하고 조회할 가계부를 선택해요. |

설명은 오해 가능성이 있는 `단톡방 연결`, `기록 방법`, `가계부 전환` 세 기능에만 사용합니다.

보조 문법은 `도움말`, `내 이름 설정`, `수정가이드`, `정산`, `초대코드`이며 대표 목록에 추가하지 않습니다. 기존 붙여쓰기 문법은 호환용으로 계속 인식합니다.
