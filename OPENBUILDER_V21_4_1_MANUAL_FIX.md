# OpenBuilder V21.4.1 수동 수정 절차

## 왜 Worker 배포만으로 끝나지 않는가
실제 카카오 응답으로 확인된 `원하는 메뉴를 골라서 바로 시작할 수 있어요` 문구는 현재 V21.4.1 Worker 소스에 없습니다. 이 문구가 계속 보이면 다음 중 하나입니다.

- OpenBuilder 블록에 고정 응답이 남아 있음
- 폴백 블록이 `/skill`이 아니라 고정 응답으로 연결됨
- Cloudflare Worker가 이전 버전으로 배포되어 있음

## 1. 배포 버전 확인
브라우저에서 다음 주소를 확인합니다.

```text
https://ttokttok-accountbook.com/health
```

응답의 `version`은 다음이어야 합니다.

```text
V21.4.1-KAKAO-RESPONSE-ROUTING-HOTFIX
```

다른 버전이면 먼저 Cloudflare Worker를 다시 배포합니다.

## 2. 폴백 블록
- 고정 말풍선 응답을 제거합니다.
- Skill을 사용하도록 설정합니다.
- Skill URL은 다음 하나만 사용합니다.

```text
https://ttokttok-accountbook.com/skill
```

자유 지출문과 등록되지 않은 자연어는 모두 폴백을 통해 `/skill`로 전달되어야 합니다.

## 3. 예산 블록
대표 발화에 다음을 등록하고 Skill을 연결합니다.

```text
남은예산
남은 예산
내 예산
우리 예산
예산 확인
예산 현황
남은 돈
얼마 남았어
```

블록 자체의 고정 응답과 URL 버튼은 제거합니다.

## 4. 정산 블록
`정산`을 요약 블록에서 제거하고 별도 블록으로 둡니다.

```text
정산
정산 현황
정산 요약
```

정산 블록도 같은 `/skill`을 호출합니다. 월 합계용 `요약` 블록과 합치지 않습니다.

## 5. 단톡방 연결 블록
다음 발화를 Skill로 전달합니다.

```text
단톡방 연결
단톡방 연결 ABC123
연결 확인
그룹 정보
```

고정 응답을 사용하지 않아야 Worker의 실제 줄바꿈 보정이 적용됩니다.

## 6. 링크·quickReplies 정책
- 일반 블록의 URL 버튼과 URL 문구를 제거합니다.
- 일반 응답의 quickReplies를 제거합니다.
- 사용자가 `링크` 또는 `주소`를 요청했을 때만 대표 주소 하나를 보냅니다.

```text
https://ttokttok-accountbook.com
```

## 7. 배포 후 실사용 테스트
개발 채널과 실제 그룹방에서 순서대로 확인합니다.

```text
@똑똑한가계부 내 예산
/단톡방 연결
/요약
/정산
/링크
점심 12000원 국민카드
```

합격 기준:
- `내 예산`이 메뉴가 아니라 예산 현황으로 응답
- `단톡방 연결`에 문자 `\n`이 보이지 않음
- `요약`과 `정산`의 내용이 서로 다름
- `/링크` 외 응답에는 URL이 없음
- 일반 응답에는 quickReplies가 없음
