# V22.8.6 영수증 화면 최적화·안정화 보고서

## 기준과 결론

V22.8.5 모바일 접근·메뉴 위계를 유지하면서 영수증 등록 화면의 선택, OCR 실행, 취소, 저장 흐름을 안정화했습니다. 분석 화면과 기존 로그인·메뉴 흐름은 변경하지 않았습니다.

## 적용 내용

- 앨범 선택과 카메라 촬영 입력 분리
- PC 드래그 앤 드롭과 클립보드 이미지 붙여넣기
- 선택 사진 미리보기와 선택 취소
- 25MB 제한과 이미지 형식 확인
- 사용자가 눌러야 시작되는 Tesseract 5.1.1 지연 로딩
- 기기 성능에 따른 1400px 또는 1800px 사전 축소
- 연속 OCR 작업자 재사용과 유휴 해제
- 전체 과정 90초 제한과 사용자 분석 취소
- 취소된 작업자가 새 실행에 섞이지 않는 실행 세대 가드
- 인식·직접 입력 금액 천 단위 쉼표
- 영수증 화면 전용 최소 조회와 요청 단위 인증 캐시
- 같은 영수증 중복 저장 및 조회 전용 권한 저장 차단

## 배포 영향

- 적용 파일: `src/index.js` 전체 교체
- 신규 SQL: 없음
- 환경변수: 변경 없음
- OpenBuilder: 변경 없음
- DB 기준: 기존 V22.8.0 스키마 유지
