# v18.8 전체 안정화 가이드

## 분석 그래프
일별 소비 그래프와 요일별 소비 추이는 인라인 안정 스타일을 적용했습니다. 일부 화면에서 CSS가 누락되어도 세로 텍스트로 깨지지 않도록 했습니다.

## 캘린더
모바일에서는 전체 날짜를 모두 펼치면 스크롤이 길어지므로, 기록 있는 날짜를 먼저 보여주고 전체 달력은 접어서 확인합니다.

## 예산
분류별 예산 합계가 월 전체 예산보다 크거나, 월 전체 예산이 수입 기준보다 크면 경고를 표시합니다.

## 정기지출
분류는 긴 드롭다운 대신 직접 입력하면서 추천 목록을 참고하는 방식으로 바꿨습니다.

## 가계부/참여자
일반 사용자는 자신이 참여한 가계부만 볼 수 있습니다. 카카오 고유키는 유지하고 가계부 안 표시 이름만 바꿉니다.
