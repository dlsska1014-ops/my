## 역할별 권한

- [ ] owner: 기록·설정·초대·권한·단톡방 연결 가능
- [ ] admin: 기록·설정·초대·권한·단톡방 연결 가능
- [ ] member: 기록 가능, 다른 구성원 관리 불가
- [ ] viewer: 조회 가능, 기록·가져오기·설정 변경 불가
- [ ] pending: 상태 안내만 표시, 데이터 조회·쓰기 불가
- [ ] blocked: 제한 안내만 표시, 데이터 조회·쓰기 불가
- [ ] 초대코드는 owner/admin에게만 표시
- [ ] 역할 제한 화면에 현재 상태·요청 대상·돌아갈 경로가 모두 표시

## 여러 가계부·단톡방
