## 권한·격리 회귀

- 참여하지 않은 사용자의 다른 가계부 영수증 저장 거부
- 동일 영수증은 같은 가계부에서만 중복 제외되고 다른 가계부에서는 독립 저장
- viewer의 설정·정산 완료 변경 거부
- 리포트 설정 키의 가계부 ID와 본문이 다르면 `invalid` 처리
- 다중 가계부·계정 통합·참여 승인·단톡방 연결 기존 회귀시험 통과

## 마이그레이션 검토

- 감사 SQL과 마이그레이션 파일 존재 및 필수 감사·인덱스·잠금 테이블·claim/release 함수·service_role 제한 정적 검사 통과
- 마이그레이션은 `begin`/`commit` 트랜잭션이며 중복이 있으면 `raise exception`으로 중단
- 기존 행을 자동 삭제하거나 변경하는 정리 SQL 없음
