# V22.5.2 변경 보고서

- 대표 명령어 10개를 V22.5.1 가계부 컨텍스트 안전 규칙과 정렬했습니다.
- `/` 단독 입력은 대표 명령어 메뉴로 처리합니다.
- 모든 선행 슬래시 명령을 기존 자연어 처리기와 동일하게 처리합니다.
- `/기록 내용`, `/수정 01번 ...`의 기존 접두어 처리도 유지합니다.
- `/kakao-command-menu.json`에 대표·보조 명령어와 등록 참고사항을 제공합니다.
- 구형 안내 문구 `오늘예산`, `분류별지출`, `새가계부 만들기 여행비`를 제거했습니다.
- OpenBuilder 시나리오명·블록명, Supabase 스키마, 거래·예산·권한·가계부 컨텍스트 안전 로직은 변경하지 않았습니다.
